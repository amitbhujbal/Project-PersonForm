# Project-PersonForm
A simple iOS app that stores and manages personal information using Core Data.

## Demo

![Demo.](https://amitbhujbal.com/images/demo-Project-PersonForm.gif "Demo")

## Features

- Create new person record
- Automatic last updated timestamp
- View record history
- Export all records as CSV file
- Persistent local storage using Core Data

## Screens

1. Person Form
   - Name
   - Birth Date
   - Education
   - Place
   - Job
   - Mobile Number
   - Email ID
   - Submit / Clear buttons

2. Record History
   - List of saved records
   - Last updated date
   - Dismiss button
   - CSV Export option


## Requirements

- iOS 15+
- Xcode 15+
